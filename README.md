## Build Pulse Panel from source:

### 1. clone this repository

```bash
git clone https://github.com/meadowsjared/pulse-panel.git
```

### 2. install packages

```bash
bun i
```

### 3. Run

### dev mode

```bash
bun dev  
```

### preview mode

```bash
bun preview  
```

### build app

```bash
bun build
```
